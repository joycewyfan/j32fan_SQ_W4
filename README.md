# Side Quest Week 4

## Setup and Interaction Instructions

To run the sketch locally, open `index.html` in Google Chrome using Live Server.

Click **Start Adventure** to begin. Click and navigate through the interactive narrative by clicking the buttons presented on each screen to make choices. Each decision leads to a different path and one of several unique endings.

After reaching an ending, click Play Again to return to the title screen and explore a different storyline.
